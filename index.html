Запуск

Создай папку nodeschool. В ней папку урока, в ней по папке на задание. Удобно именовать папки заданий с номером. Например, 0-hello-world.

1. Открой консоль и перейди в созданную папку первого задания
2. Открой список воркшопов (nodeschool.io/#workshopper-list) и выбери, чему хочешь научиться
3. Установи модули уроков, которые будешь проходить
4. Для запуска воркшопа напиши его название в консоли
